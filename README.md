## Preview docs locally

```bash
# preview mkdocs
mkdocs serve
```

## Manually publish to GitHub Pages

Deployment is done automatically once on master.

Instructions for [deploying in GitHub](http://www.mkdocs.org/user-guide/deploying-your-docs/):

```bash
mkdocs gh-deploy
```
