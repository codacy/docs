## Bug fixes

-   The [Codacy configuration file](../../repositories-configure/codacy-configuration-file.md) no longer overrides the [default ignored files](../../repositories-configure/ignoring-files.md#default-ignored-files). Also, Codacy no loger ignores by default files matching the regular expression `.*docs?/.*`. (IO-1031)
